# 쇼타로 춤 소환하기 🪄

이모지를 조합해 쇼타로(RIIZE)의 춤 챌린지를 소환하는 웹 게임.

- 이모지 **1~5개**를 골라 소환 → 조합마다 **다른 춤 릴스(유튜브)** 가 재생 (총 31조합)
- **같은 조합 = 같은 결과** (공유 링크로 재현 가능)
- 모바일 최적화 · 음소거 자동재생 + 탭하여 소리 켜기 · **X 공유**

## 로컬 실행

정적 사이트라 아무 웹서버로 열면 됩니다.
유튜브 임베드는 **http(s)에서만** 동작하니 `file://`(더블클릭)로는 열지 마세요.

```bash
npx serve .
# 또는
python3 -m http.server 8000
```
